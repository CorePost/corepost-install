corepost-install регистрирует устройство и подготавливает локальную конфигурацию системы.
Скрипт получает provisioning bundle, пишет конфиг preboot, разворачивает agent и обновляет initramfs.
Поддерживается подготовка USB-фактора для профиля 3FA и локальный demo flow.
Компонент работает как root на Debian-совместимой системе с curl, openssl и initramfs-tools.
Все пути, репозитории, ref и серверные параметры переопределяются через окружение.
